# Proyectos

¡Buenas gente! 👋

Se me ocurrió que podíamos armar un proyecto colaborativo entre nosotros. La idea es simple: juntarnos para practicar, darnos una mano con el código y terminar con algo real que todos podamos subir a nuestro portfolio.

¿Cómo nos organizamos?

Este finde: Vayan tirando ideas por acá. Cualquier cosa que tengan ganas de programar o probar, suéltenla sin miedo.

El lunes (20/04): Nos juntamos en una sala de estudio para elegir qué idea seguir y ver cómo nos repartimos.

Horarios de la reunión:

🇪🇸 España: 21:00hs

🇦🇷 Argentina / 🇨🇱 Chile: 16:00hs

🇲🇽 México / 🇨🇴 Colombia / 🇵🇪 Perú: 13:00hs

No importa si recién están arrancando con la programación o si ya saben más; la idea es aprender entre todos y que nadie se quede afuera.

¡Cualquier aporte suma!
